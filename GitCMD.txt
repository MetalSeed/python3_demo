# repo init
git init


# repo commit
git add .
git commit -m "str"


# commit status
git status


# code version
git log
git reflog


# code backtrack
git reset --hard HEAD^
git reset --hard commit_id

//
HEAD指向的版本就是当前版本，因此，Git允许我们在版本的历史之间穿梭，使用命令git reset --hard commit_id。

穿梭前，用git log可以查看提交历史，以便确定要回退到哪个版本。

要重返未来，用git reflog查看命令历史，以便确定要回到未来的哪个版本。
//
